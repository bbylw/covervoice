# 9:16 竖版配图提示词

用于 Codex 环境下为本 skill 生成短视频封面配图。提示词只负责定基调，不要写成长篇说明。先判断图片落位，再选择类型。

## 通用规则

- 风格基调：电子杂志 × 电子墨水，克制、真实、留白充足，适合 9:16 竖版封面
- 图片中的文字语言必须跟随用户语言：中文封面用中文，英文封面用英文
- 不生成卡通、3D、霓虹科技感、SaaS 模板感、过度装饰或假 logo
- 图片要给标题留出可叠加空间，不要满屏堆细节
- 封面配图是背景或视觉素材，不是独立设计稿：不要生成页眉、页脚、角标、署名、装饰边框
- 生成后保存到 `images/`，命名为 `{语义}.{ext}`（封面通常只用 1-3 张图）

## 尺寸选择

| 用途 | 推荐尺寸 | HTML 落位 |
|------|---------|-----------|
| 全幅背景图 | 1080 x 1920 | `.layout-overlay` 中的全幅 `frame-img` |
| 上半视觉图 | 1080 x 900 | `.layout-split-v` 中的 `.visual-area` |
| 方形装饰图 | 800 x 800 | 内联 `style="width:800px;height:800px"` |
| 竖版人物/产品 | 900 x 1200 | 灵活放置 |

## 图片标准化策略

### A. 先选目标槽位

不要先生成图片再硬塞进封面。先决定图片落位：

1. 全幅背景：1080 x 1920，需要整体偏暗或偏亮（配合主题色）
2. 上半视觉：约 1080 x 800-1000，需要竖版构图
3. 装饰小图：正方形或竖版，辅助视觉

### B. 用户原始图片的处理

原始图片比例通常不可控，按下面顺序处理：

1. 如果原图接近 9:16 竖版，直接作为背景图使用
2. 如果原图是横版，优先裁剪为竖版（保留核心主体），或重新生成
3. 如果原图比例奇怪，重新生成到 9:16 或 1:1
4. 如果必须保留原图，用 `object-fit:contain` 放进 frame，接受留白

### C. 生成提示词后缀

每个配图提示词最后都补一句规格约束：

```text
输出必须是 9:16 竖版构图（1080x1920），主体居中但保留安全边距（上下左右至少留 80px 空白给文字叠加），画面密度中等，不要生成文字、水印、logo、页眉、页脚、角标、装饰边框或横向构图。
```

如果是方形图：

```text
输出必须是 1:1 正方形构图，主体居中，画面密度中等，不要生成文字、水印、logo、页眉、页脚、角标、装饰边框。
```

## 类型 1: 竖版抽象纹理/渐变

用于暗色/亮色主题的封面背景，不抢文字注意力。

```text
生成一张 9:16 竖版抽象纹理背景图，风格为电子墨水杂志风，细腻颗粒、柔和渐变、低饱和中性色调（深灰/暖米/靛蓝/森林绿方向），无明显中心焦点，整体均匀分布，适合作为文字叠加背景。不要生成文字、水印、logo、具体物体或人物。画面密度低，留白充足。输出必须是 9:16 竖版构图（1080x1920），上下左右至少留 80px 安全边距。
```

## 类型 2: 竖版人物肖像

用于有主讲人/人物的内容封面。

```text
生成一张 9:16 竖版人物肖像照片，风格像 Fujifilm / Leica editorial documentary，自然光、低饱和、轻微胶片颗粒。人物上半身或面部特写，背景简洁虚化。适合电子杂志风短视频封面，给顶部和底部留出文字叠加空间。不要商业摆拍感、不要过度修图、不要 logo 或水印。输出必须是 9:16 竖版构图（1080x1920），主体偏中上，底部留至少 200px 暗色/简洁区域给文字。
```

## 类型 3: 竖版场景/氛围照

用于生活方式、旅行、文化类封面。

```text
生成一张 9:16 竖版场景氛围照片，主题是：[场景描述]。风格像独立杂志 editorial photography，自然光、低饱和、真实质感、克制留白。适合作为短视频封面背景，需要给文字叠加留出空间。不要过度饱和、不要游客照、不要 logo 或水印。输出必须是 9:16 竖版构图（1080x1920），画面主体偏中上，底部留至少 200px 相对简洁/暗色区域。
```

## 类型 4: 竖版产品/物品特写

用于产品评测、好物分享类封面。

```text
生成一张 9:16 竖版产品特写照片，主体是：[产品名称]。风格像高端独立杂志产品摄影，柔和侧光、低饱和、细腻质感、背景极简。产品居中偏上，给底部留出文字空间。不要过度打光、不要商业广告感、不要 logo 或水印。输出必须是 9:16 竖版构图（1080x1920），背景简洁，底部至少 200px 留白。
```

## 类型 5: 竖版信息图/流程图

用于知识类、教程类封面（较少用，通常直接文字排版）。

```text
生成一张 9:16 竖版杂志风信息图，解释：[概念/流程]。电子墨水风格，黑白灰为主，少量低饱和强调色，细线条、网格、编号、短标签、留白充足。图中文字使用[中文/英文]，保持简短可读。不要卡通、3D、霓虹科技感或模板感。输出必须是 9:16 竖版构图（1080x1920），主体居中，不要页眉、页脚、角标、署名或装饰边框。
```

## 类型 6: 竖版数据大字报视觉

用于成绩展示、里程碑封面。

```text
生成一张 9:16 竖版数据大字报视觉背景，风格为电子墨水杂志版式，超大衬线数字占位（仅作为视觉纹理，不要可读数字）、细线、网格、留白和纸张质感。不要生成可读的文字内容、不要 logo 或水印。输出必须是 9:16 竖版构图（1080x1920），整体偏暗/偏亮（根据主题），给中心区域留足文字叠加空间。
```

## 推荐选择参考

| 如果是... | 推荐图片类型 |
|---|---|
| 暗色主题 + 无特定视觉 | 类型 1: 抽象纹理 |
| 有人物出镜 | 类型 2: 人物肖像 |
| 生活方式/旅行/文化 | 类型 3: 场景氛围 |
| 产品/好物 | 类型 4: 产品特写 |
| 教程/知识/流程 | 类型 5: 信息图（或不用图） |
| 数据/成绩/里程碑 | 类型 6: 数据大字报背景 |
